# Longest palindromic substring

Given a string s, return the longest palindromic substring in s. A palindromic substring is a substring that is a palindrome.

Example

```,
Input - "programming"
Output - "mm"

Input - "bananas"
Output - "anana"

Input - "xabax"
Output - "xabax"

Input - "mind"
Output - "m"
```
